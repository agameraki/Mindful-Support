# 🌿 Mindful Support

## 🧠 Problem Statement

Mental health issues are becoming increasingly common, yet many individuals hesitate to seek help due to stigma, lack of accessibility, or difficulty expressing their emotions. People often struggle to consistently track their mood, reflect on their thoughts, and find reliable mental health resources in one place.

**Mindful Support** aims to provide a safe digital environment where users can monitor their emotional well-being through mood tracking, personal journaling, and guided mental wellness resources.

---

## 🛠️ Tech Stack

* **Frontend:** EJS, HTML, CSS, JavaScript
* **Backend:** Node.js, Express.js
* **Database:** MongoDB (Mongoose)
* **Real-Time Communication:** Socket.io
* **Charts & Analytics:** Chart.js
* **Environment Management:** dotenv

---

## ✨ Features

### 🔐 Secure Authentication

* User **signup and login system**
* Protected routes using **authentication middleware**
* Secure session handling

### 😊 Mood Tracking

* Users can log their **daily mood**
* Helps track emotional patterns over time

### 📊 Mood Analytics

* Visual mood trends using **Chart.js**
* Graphical representation of mood history for better understanding

### 📓 Personal Journaling

* Users can create **private journal entries**
* Store and revisit past reflections

### 🧘 Mental Health Resources

The platform provides curated mental wellness resources including:

* Guided **breathing exercises**
* **Meditation support**
* **Crisis help information**

### 💬 Real-Time Support Chat

* Implemented using **Socket.io**
* Enables real-time communication support

### 🎨 Theme Switching

* Users can switch between different **themes for a better user experience**

### 📱 Responsive Design

* Fully responsive interface
* Works across **desktop, tablet, and mobile devices**

### ⚙️ User Settings

* Allows users to manage personal preferences
